
jmolwidgets
=====================================

Version: |release|

A custom Jupyter Lab widget for Jmol/Jsmol


Quickstart
----------

To get started with jmolwidgets, install with pip::

    pip install jmolwidgets

or with conda::

    conda install jmolwidgets
